handbook.cs.virginia.edu

This page contains the undergraduate and graduate handbooks for the computer science department.

The undergraduate handbook is available as a PDF; this is always the latest version. The source is available in the uva-cs/ugrad-handbook github repo. The relevant pages on the UVa CS website can be found here.

The graduate handbook is available as a PDF; this is always the latest version. The source is available in the uva-cs/grad-handbook github repo. The relevant pages on the UVa CS website can be found here.

The primary author of this handbook is Aaron Bloomfield (aaron@virginia.edu, github.com/aaronbloomfield). If you have any suggestions, corrections, etc., please either execute a pull request or contact him.
